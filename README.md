# ChartFrame — 图表装裱工具

一个轻量级的网页工具，用于给数据图表添加标题、说明、水印，并导出为高清图片。

## 使用方法

直接用浏览器打开 `index.html` 即可使用，无需安装任何依赖。

### 基本流程

1. **导入图表** — 拖入图片 / Ctrl+V 粘贴截图 / 点击「选择文件」
2. **填写标注** — 标题、副标题、注释、底部说明
3. **添加水印**（可选）— 支持文字水印和图片水印
4. **导出** — 点击「导出 PNG」或「导出 JPG」，默认保存到下载文件夹

### 功能特性

- **实时预览** — 所有修改即时反映在右侧预览区
- **高清导出** — 支持 1x–4x 倍率，默认 4x
- **文字水印** — 输入文字自动生成平铺水印，内容自动记忆
- **图片水印** — 上传 Logo 等图片作为水印，可保存为默认设置
- **水印参数** — 可调节大小（百分比）、透明度、旋转角度、间距
- **字体支持** — 优先使用 OPPO Sans 4.0（本机安装或上传字体文件）
- **另存为** — 支持选择自定义保存位置

### 浏览器兼容

推荐使用 Chrome / Edge 等基于 Chromium 的浏览器，可获得完整功能（包括「另存为」选择保存位置）。

## 外部依赖

- [html2canvas v1.4.1](https://cdnjs.cloudflare.com/ajax/libs/html2canvas/1.4.1/html2canvas.min.js) — 用于将 DOM 渲染为 Canvas 并导出图片（通过 CDN 加载）
